Questions générales

Qu'est-ce que PAYTOU ?

PAYTOU est un portefeuille électronique qui vous permet de faire des transferts d'argent vers n'importe quel porte-monnaie électronique. Vous pouvez effectuer des dépôts, des retraits et des transferts d'argent entre utilisateurs PAYTOU gratuitement.

Comment fonctionne PAYTOU ?

PAYTOU fonctionne comme un portefeuille électronique. Après avoir créé un compte et chargé votre solde, vous pouvez envoyer de l'argent à d'autres utilisateurs, payer chez des commerçants partenaires, ou retirer des espèces. Toutes les transactions sont instantanées et sécurisées.

Où puis-je télécharger l'application PAYTOU ?

L'application PAYTOU est disponible gratuitement sur Google Play Store pour les utilisateurs Android et sur App Store pour les utilisateurs iOS. Recherchez simplement "PAYTOU" dans votre magasin d'applications préféré.

Dans quels pays PAYTOU est-il disponible ?

PAYTOU est actuellement disponible dans [liste des pays]. Nous travaillons constamment à étendre notre service à de nouveaux pays. Restez à l'écoute pour les annonces d'expansion.

Questions sur les comptes

Comment créer un compte PAYTOU ?

Pour créer un compte PAYTOU, téléchargez l'application, puis suivez ces étapes :

  1. Cliquez sur "Créer un compte"
  2. Renseignez votre numéro de téléphone
  3. Entrez le code de vérification reçu par SMS
  4. Complétez votre profil avec les informations requises
  5. Choisissez votre code PIN de sécurité
  6. Votre compte est prêt à être utilisé !

Quels documents sont nécessaires pour créer un compte ?

Pour créer un compte PAYTOU, vous aurez besoin d'une pièce d'identité valide (carte d'identité, passeport ou permis de conduire) et d'un numéro de téléphone mobile actif. Pour certains niveaux de compte, une preuve de résidence (facture récente d'électricité, d'eau ou de téléphone) peut être demandée.

Comment modifier mes informations personnelles ?

Pour modifier vos informations personnelles :

  1. Connectez-vous à votre compte PAYTOU
  2. Accédez à la section "Profil" depuis le menu
  3. Cliquez sur "Modifier" à côté des informations que vous souhaitez mettre à jour
  4. Effectuez vos modifications et confirmez

Notez que certaines modifications peuvent nécessiter une vérification supplémentaire pour des raisons de sécurité.

Puis-je avoir plusieurs comptes PAYTOU ?

Non, chaque utilisateur ne peut avoir qu'un seul compte PAYTOU associé à son numéro de téléphone et à sa pièce d'identité. Cette politique est en place pour garantir la sécurité et la conformité aux réglementations financières.

Questions sur les transactions

Comment envoyer de l'argent avec PAYTOU ?

Pour envoyer de l'argent :

  1. Ouvrez l'application PAYTOU et connectez-vous
  2. Cliquez sur "Envoyer" depuis l'écran d'accueil
  3. Sélectionnez un contact ou entrez le numéro de téléphone du destinataire
  4. Saisissez le montant à envoyer
  5. Vérifiez les détails et confirmez avec votre code PIN
  6. Le destinataire recevra immédiatement l'argent dans son compte PAYTOU

Quels sont les frais pour les transferts d'argent ?

Les transferts entre utilisateurs PAYTOU sont totalement gratuits. Pour les transferts vers d'autres services ou les retraits, des frais peuvent s'appliquer. Consultez notre page Tarifs pour obtenir les détails complets.

Combien de temps prend un transfert d'argent ?

Les transferts entre utilisateurs PAYTOU sont instantanés. Le destinataire reçoit l'argent immédiatement, dès que vous confirmez la transaction. Les deux parties reçoivent une notification instantanée pour confirmer la réussite de l'opération.

Comment déposer de l'argent sur mon compte PAYTOU ?

Vous pouvez déposer de l'argent sur votre compte PAYTOU de plusieurs façons :

  • Chez un agent PAYTOU autorisé
  • Par transfert bancaire
  • Via un autre service de paiement mobile compatible
  • Par carte bancaire directement dans l'application

Pour plus de détails sur chaque méthode, consultez la section "Dépôt" dans l'application.

Questions sur la sécurité

Comment PAYTOU protège-t-il mes informations ?

PAYTOU utilise un cryptage de pointe pour protéger toutes vos informations personnelles et financières. Nous mettons en œuvre des mesures de sécurité multiples, notamment :

  • Authentification à deux facteurs
  • Cryptage des données de bout en bout
  • Surveillance continue des transactions pour détecter les activités suspectes
  • Protections contre la fraude
  • Verrouillage automatique de l'application après une période d'inactivité

Nous sommes également conformes aux normes internationales de sécurité des données financières.

Que faire si j'oublie mon code PIN ?

Si vous oubliez votre code PIN :

  1. Sur l'écran de connexion, cliquez sur "Mot de passe oublié"
  2. Suivez le processus de vérification d'identité
  3. Vous recevrez un code de réinitialisation par SMS
  4. Créez un nouveau code PIN

Pour des raisons de sécurité, certaines fonctionnalités de votre compte peuvent être temporairement limitées après une réinitialisation de PIN.

Que faire si je perds mon téléphone ?

Si vous perdez votre téléphone :

  1. Contactez immédiatement notre service client au +XX XX XX XX XX
  2. Demandez le blocage temporaire de votre compte
  3. Une fois que vous avez un nouveau téléphone, téléchargez à nouveau l'application
  4. Connectez-vous avec vos identifiants et suivez le processus de vérification
  5. Votre compte sera réactivé sur votre nouvel appareil

Pour une sécurité optimale, changez votre code PIN après avoir récupéré l'accès à votre compte.

Questions pour les commerçants

Comment devenir commerçant partenaire PAYTOU ?

Pour devenir commerçant partenaire PAYTOU :

  1. Remplissez le formulaire de demande en ligne sur notre site web
  2. Fournissez les documents requis pour votre entreprise
  3. Notre équipe commerciale vous contactera pour finaliser le processus
  4. Après approbation, vous recevrez votre kit marchand PAYTOU
  5. Notre équipe vous aidera pour l'installation et la formation

Pour plus d'informations, contactez notre service commercial à merchant@paytou.com

Quels sont les avantages pour les commerçants ?

Les commerçants partenaires PAYTOU bénéficient de nombreux avantages :

  • Frais de transaction réduits par rapport aux TPE traditionnels
  • Règlement des paiements plus rapide
  • Aucun coût d'équipement (simple QR code à afficher)
  • Dashboard marchand pour suivre toutes vos transactions
  • Assistance technique prioritaire
  • Visibilité auprès de notre base d'utilisateurs grandissante
  • Réduction des manipulations d'espèces

Comment fonctionne le paiement par QR code ?

Le paiement par QR code est simple et rapide :

  1. Le commerçant génère un QR code avec le montant de la transaction
  2. Le client scanne ce QR code avec l'application PAYTOU
  3. Le client vérifie le montant et confirme le paiement avec son code PIN
  4. Le commerçant reçoit instantanément une notification de paiement
  5. La transaction est terminée et le montant est crédité sur le compte marchand

Vous pouvez également générer un QR code statique pour votre boutique, et le client pourra saisir le montant lui-même.
